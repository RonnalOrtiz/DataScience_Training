# 🗄️ SQL Training

data analysis, using PostgreSQL and the Brazilian E-Commerce
(Olist) dataset.

## 🧠 Skills Practiced

- Day 1: `SELECT`, `FROM`, `WHERE`, `ORDER BY`
- Day 2: `GROUP BY`, `HAVING`, `Aggregations`
- Day 3: `JOIN` types (INNER, LEFT, RIGHT, FULL)
- Day 4+: Subqueries, CTEs, Window Functions
